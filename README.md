# DragonLife - Math Game

A web-based math game for kids and teenagers. Earn coins by completing math challenges!

## Development

```bash
npm install
npm run dev
```

## Features

- Touch-friendly UI (tablet-ready)
- Username-based signup with grade selection
- Coin reward system
- Multiple choice math challenges
- Progressive difficulty
- Categories: Arithmetic, Algebra, Geometry, Fractions, Probability, Trigonometry
